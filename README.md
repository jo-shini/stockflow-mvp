# StockFlow MVP
 
## Tech Stack
- Backend: Node.js + Express
- Frontend: React
- Styling: Tailwind CSS
- Database: MongoDB
- Auth: JWT
 
## Start Time
8:30PM
 
## Description
Minimal SaaS inventory management system with authentication, product CRUD, and dashboard.
 
## Features (MVP)
- User Signup & Login
- Organization-based data isolation
- Product CRUD (Create, Read, Update, Delete)
- Dashboard with inventory summary
- Low stock tracking
- Default low stock threshold (settings)
